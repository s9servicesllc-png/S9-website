# S9 Services LLC Website

Official website for **S9 Services LLC**, a Michigan-based exterior construction company providing roofing, siding, decks, windows, doors, and residential building services.

This project is a **clean single-page HTML website** designed to be fast, professional, and easy to deploy.

---

# Website Features

The website includes the following sections:

• Hero landing section  
• About S9 Services LLC  
• Services overview  
• Why Choose Us  
• Recent Projects gallery  
• Customer Reviews / Testimonials  
• Insurance Claims Process (3 steps)  
• FAQ section  
• Contact / Project Request section  

The site uses a **clean white design** so the S9 Services logo stands out clearly.
